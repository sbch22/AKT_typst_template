# Audiokommunikation und -technologie – Typst Template für Masterarbeiten

Dieses Repository stellt ein **Typst-Template** für Abschlussarbeiten im Fachgebiet *Audiokommunikation und -technologie* an der TU Berlin bereit. Es unterstützt sowohl **einspaltige** als auch **zweispaltige** Layouts im offiziellen TU-Berlin-Stil.

## Verwendung

Das Layout kann direkt in der Datei `main.typ` angepasst werden. Dort lässt sich unkompliziert zwischen ein- und zweispaltigem Format wechseln.

## Sprache anpassen

Um das Template auf Englisch umzustellen:

1. Öffne die Dateien `main.typ` und `template.typ`
2. Suche nach deutschen Begriffen (z. B. Kapitelüberschriften, Labels)
3. Übersetze diese entsprechend ins Englische

> Tipp: Nutze die Suchfunktion deines Editors, um alle relevanten Stellen schnell zu finden.

## Hinweise

- Das Template dient als Ausgangspunkt und kann je nach Anforderungen angepasst werden.
- Prüfe vor Abgabe immer die formalen Vorgaben deines Instituts.
